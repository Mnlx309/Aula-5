Exercício com Arrays Multidimensionais em PHP

Descrição detalhada do exercício

O objetivo deste exercício é criar um sistema simples para gerenciar informações de alunos e suas notas, utilizando arrays multidimensionais em PHP. Você deverá criar uma estrutura onde cada aluno seja representado por um array associativo contendo:

  • Nome do aluno (string)
  • Notas (array numérico contendo as três notas do aluno)

A lista completa de alunos será armazenada em um array multidimensional, onde cada elemento é um aluno com seu nome e suas notas.

Exemplo de array multidimensional esperado:

<?php
$alunos = [
    [
        "nome" => "Ana",
        "notas" => [8, 7, 9]
    ],
    [
        "nome" => "Carlos",
        "notas" => [6, 9, 7]
    ],
    [
        "nome" => "Beatriz",
        "notas" => [10, 10, 9]
    ]
];
?>

Seu programa deverá permitir que o usuário:

  • Adicione alunos e suas três notas
  • Liste todos os alunos, mostrando o nome, as notas (separadas por vírgula) e a média das notas
  • Calcule a média de cada aluno somando as notas com array_sum() e dividindo pela quantidade de notas, obtida com count()

Para entrada de dados no terminal, utilize a função readline(), que permite capturar as informações digitadas pelo usuário. Para exibir as notas separadas por vírgula, utilize a função implode(), que junta os elementos de um array em uma string.

Resumo das funções usadas no exercício:

  • readline(): captura dados digitados pelo usuário no terminal.
  • array_sum(): soma todos os valores de um array numérico.
  • count(): retorna o número de elementos de um array.
  • implode(): junta elementos de um array em uma string, separados por um caractere especificado.

Exemplos simples das funções:

1. readline()

Lê uma entrada do usuário no terminal.

<?php
$nome = readline("Digite seu nome: ");
echo "Olá, $nome!";
?>

2. array_sum()

Soma os valores dentro de um array.

<?php
$notas = [7, 8, 9];
$total = array_sum($notas);
echo "Total das notas: $total"; // Saída: 24
?>

3. count()

Conta quantos elementos existem em um array.

<?php
$notas = [7, 8, 9];
$quantidade = count($notas);
echo "Quantidade de notas: $quantidade"; // Saída: 3
?>

4. implode()

Une elementos de um array em uma string separada por vírgulas.

<?php
$notas = [7, 8, 9];
echo implode(", ", $notas); // Saída: 7, 8, 9
?>

5. strtolower()

Converte uma string para minúsculas.

<?php
$texto = "PHP É MUITO LEGAL";
echo strtolower($texto); // Saída: php é muito legal
?>

6. number_format()

Formata um número com casas decimais.

<?php
$numero = 12.3456;
echo number_format($numero, 2); // Saída: 12.35
?>
